## Installation

1. Clone the repository to your local machine:

   ```bash
   git clone https://github.com/Exaster/ISA.git
   ```

2. Open the project in Flutter:

   ```bash
   cd ISA
   flutter pub get
   ```

3. Run the app on your device:

   ```bash
   flutter run
   ```
